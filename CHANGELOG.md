# Changelog

此文件記錄 **CSI Portal** 專案的所有重大變更。
版本號遵循 [語義化版本 (SemVer)](https://semver.org/lang/zh-TW/) 規範：
* **Major (X)**：重大架構改動（如更換核心框架）。
* **Minor (Y)**：新增功能。
* **Patch (Z)**：Bug 修復或效能優化。

---

## [1.11.0] - 2026-04-07
### Added (新增)
* **Tube Hub 影音中樞與智慧解析引擎**：
    * 雙軌下載與自動字幕解析：深度整合 yt-dlp 套件，支援從前端直接搜尋並無縫下載 YouTube 資源。系統提供「影片類型」與「音樂類型」雙模式，並透過 youtube_transcript_api 自動抓取原生字幕或進行多國語言的即時翻譯轉換，結合個人筆記打造全方位學習載體。
    * 階層式收藏夾與分類系統：實作高度客製化的個人化資料夾架構，前端採用 Bootstrap 手風琴介面渲染，支援資源在不同資料夾與根目錄間自由移動，讓個人知識庫與影音清單井然有序。

* **Web Audio 即時混音與 KTV 練唱系統**：
    * 純前端即時升降 Key 處理：突破原生 HTML5 限制，於播放器底層導入 Tone.js 函式庫接管音訊路由。利用 Web Audio API 實現即時的音高位移，讓玩家在練唱時能針對伴奏進行正負 12 半音的升降 Key 調整，且完全不影響原曲播放節奏。
    * 專業學習播放控制器：整合 0.5x 至 2.0x 無段語速調整、快捷快進/倒退，以及專為外語學習與歌曲副歌拆解設計的「A-B 段落重複播放」功能，並支援全曲循環，滿足深度練唱與聽力訓練需求。
    * Three.js 3D 沉浸式背景引擎：捨棄傳統 2D CSS 動畫，導入 Three.js 打造立體視角的飄落粒子系統。透過客製化 Shader 材質與物理搖曳運算，在深色背景下營造具備空間縱深感的練唱氛圍。

* **社群共享生態與一鍵收藏機制**：
    * 資源權限動態切換：賦予使用者「公開/私有」的單件資源控制權，可隨時將優質的影片或音樂分享至社群大廳。
    * 零成本檔案共用技術：社群大廳內的公開資源支援「一鍵收藏」。當玩家收藏他人分享的資源時，系統僅會在資料庫建立關聯紀錄，實體影音檔案則完美共用，大幅降低伺服器硬碟負載。

### Changed (變更)
* **實體檔案生命週期與儲存空間最佳化**：
    * 智慧型垃圾回收：重構資源刪除邏輯。當使用者移除個人收藏時，系統會自動核對資料庫的引用狀態。若該實體 .mp4 或 .mp3 檔案已無任何使用者綁定，系統將從後端自動清除該實體檔案，實現伺服器容量的最佳化釋放。
    * 高對比度 UI 與操作防呆：優化深色模式下的視覺焦點，將主要行動按鈕 (如「收藏」) 重構為發光警示色並加入立體懸浮特效。同時在播放器載入時強制監聽全域點擊事件，確保 Web Audio Context 能合法啟動，避免瀏覽器安全性阻擋音頻引擎。

## [1.10.1] - 2026-03-31
### Added (新增)
* **全方位 AI 影像視覺中樞**：
    * YOLOv11 物件偵測整合：擴充原有的 OCR 辨識頁面，於 AI 引擎選單新增「YOLOv11」選項，將單純的文字辨識系統正式升級為多模態的視覺分析平台。
    * 即時視覺化標註與統計：當選擇 YOLO 引擎時，後端會自動在圖片上繪製物件辨識方框，並透過 Base64 技術將左側預覽圖無縫替換為「AI 標註影像」。右側文字區塊則會動態生成對齊完善的「物件偵測統計表」，各類物品總數一目了然。
    * 前端防呆與聯動邏輯：導入智慧型表單控制腳本，當使用者切換至物件偵測模式時，系統會自動反灰並鎖定「語言選擇」下拉選單，避免邏輯衝突與無效操作。

* **倖存者遊戲暫停與戰力監控系統**：
    * 全局暫停控制機制：於遊戲畫面右上角置入實體的懸浮暫停按鈕，並支援電腦端玩家使用鍵盤 P 鍵或 Esc 鍵進行快捷操作。暫停時精準凍結所有生怪、位移、子彈與計時器運算。
    * 戰力疊加明細面板：於暫停畫面的覆蓋層中，精心設計了「戰力組成明細表」。系統會在底層嚴格追蹤玩家的「基礎數值」、「局外強化」與「局內升級」，讓玩家隨時掌握當下血量、攻擊、速度等能力的總和來源。

### Changed (變更)
* **原生滿版自適應與行動端觸控優化**：
    * 生 Canvas 響應式渲染：徹底捨棄 CSS Transform Scale 的強制縮放舊思維，改採監聽 window.innerWidth/innerHeight 動態重設畫布真實解析度。完美解決手機直立螢幕導致的字體過小與模糊問題，實現真正的 App 級滿版沉浸體驗。
    * 觸控事件穿透與層級修護：重構暫停選單內的按鈕綁定邏輯，建立封裝的 bindMenuButton 工具函式。透過同時監聽 click 與 touchend 事件，搭配 stopPropagation() 與 z-index 層級提升，徹底解決行動裝置上按鈕點擊無效、或誤觸底層虛擬搖桿的痛點。

* **遊戲底層屬性架構重構**：
    * 移速與攻速獨立運算：精細化「速度」天賦的定義，將籠統的屬性徹底拆分為「移動速度」與「攻擊速度」。
    * 動態冷卻時間系統：重構武器射擊邏輯，攻擊速度改以百分比基準（100% 起跳）進行計算，並動態介入武器冷卻公式（基礎 CD / 攻擊速度比例），讓攻速天賦的升級反饋更加平滑且符合直覺。

## [1.10.0] - 2026-03-30
### Added (新增)
* **沉浸式隱藏遊戲中心**：
    * 彩蛋進入機制：於 AI 客服圖示導入「長按 3 秒鎖定」邏輯，需通過密碼驗證方可切換至隱藏遊戲維度。
    * App 化獨立佈局：捨棄傳統網頁導覽列，採用全螢幕黑化沉浸式 UI，並透過 JavaScript 實作 transform: scale 自適應縮放技術，確保在任何解析度下皆能維持 800x600 的黃金遊戲比例。
    * 全域浮動虛擬搖桿：針對行動裝置開發「無形化觸控追蹤系統」，支援畫面上任意位置按下並滑動進行 360 度平滑位移，徹底解決傳統固定搖桿遮擋視線與操作手感不佳的痛點。

* **資料庫驅動的關卡與圖鑑系統**：
    * 動態關卡管理：新增 SurvivorLevel 模型，支援後端即時調整通關時限、生怪頻率、難度倍率與破關獎勵金幣。
    * 擴展性怪物圖鑑：導入 SurvivorMonster 模型，支援自定義怪物屬性（血量、防禦、速度、大小）與小縮圖上傳，並實作 Canvas ctx.clip() 圓形裁切技術，將上傳圖片完美轉化為遊戲敵機頭像。
    * 個人化角色渲染：系統自動串接 Django profile 大頭貼，若玩家已設定個人頭像，將自動取代預設綠點，成為場上唯一的倖存者。

### Changed (變更)
* **Roguelike 隨機升級機制優化**：
    * 三選一隨機天賦池：捨棄固定升級邏輯，改採隨機抽取機制。每次升級由系統自天賦池（補血、攻擊、攻速、雙發機率、移速、血量上限）抽籤產出三個選項，提升每場遊戲的策略多變性。
    * 多重射擊判定系統：重構 autoAttack 邏輯，新增 multishotChance 屬性。當觸發機率判定時，玩家將瞬間噴發第二發具備角度偏移的子彈，大幅強化中後期火力壓制感。

* **遊戲節奏與穩定性修護**：
    * 精準時間校正邏輯：修復過去進入升級或暫停選單後存活時間歸零的 Bug。透過記錄 pauseStartTime 並在返回遊戲時計算 pauseDuration 進行 startTime 平移，確保計時器與生怪節奏完美接續。
    * 效能阻塞防護：優化碰撞偵測演算法，改用距離平方比較以節省 Math.sqrt 計算開銷；並實作 hitCooldown 受傷無敵冷卻與死掉怪物的 filter 強制回收機制，有效杜絕紅球過多導致的瀏覽器卡死現象。
    * 即時互動控制：於畫面右上角置入「懸浮暫停按鈕」，支援 Esc 或 P 鍵快速切換，並同步呼叫後端 API 進行即時金幣結算與存活紀錄存檔。

## [1.8.2] - 2026-03-16
### Added (新增)
* **個人化不重複刷題系統**：
    * 作答軌跡追蹤：擴充 QuizRecord 資料庫模型，新增 attempted_questions 欄位，精準記錄使用者於每份教材中挑戰過的題目歷程。
    * 智慧題庫過濾：進入專屬學習室時，後端會自動比對歷史紀錄並隱藏已作答舊題，確保每次測驗都能專注於全新考點。
    * 破關成就機制：當該份教材的所有題庫皆被刷完時，系統將動態隱藏測驗表單，並渲染專屬的「恭喜破關」獎牌畫面，提供「一鍵呼叫 AI 擴充新題庫」的無縫導引。

* **高自由度學習控制台**：
    * 舊題顯示切換開關：於測驗頁面上方新增精美的切換開關，支援透過 ?show_all=true 網址參數動態重載，將「專注新題」或「全卷複習」的決定權交還給使用者。
    * 視覺化作答標籤：開啟全卷顯示模式時，系統會貼心地在曾經做過的題目旁，標註灰色的「<i class="fas fa-history"></i> 已作答」微型徽章，新舊考題狀態一目了然。

### Changed (變更)
* **無縫接軌的使用者體驗**：
    * 回程車票路由機制：優化 generate_analysis 視圖與前端分頁邏輯。透過隱藏欄位傳遞 next_url 與 ?tab=quiz 參數，解決過去呼叫 AI 擴充題目後被迫跳回首頁的痛點。現在訓練完畢將自動重整，並無縫停留在「練習測驗」頁籤，讓使用者立刻挑戰熱騰騰的新題目。
    * 懸浮置頂進度與強制完卷：重構測驗區塊 UI，導入 position: sticky 打造始終懸浮於畫面頂端的「作答進度面板」（包含總題數、已答、未答）。並實作嚴格的前端防呆邏輯：提交按鈕預設為鎖定反灰狀態，需待「未答題數」精準歸零後方可解鎖交卷。

* **前端架構重構與 API 防護**：
    * 腳本模組化與快取優化：將分散於 HTML 樣板中的 Markdown 即時渲染與互動邏輯，全面抽離並整合至外部的 study_brain.js 中。不僅大幅提升樣板整潔度，更善用瀏覽器快取機制加速網頁載入。
    * 擴充題庫防連點保護：針對耗時較長的「呼叫 AI 擴充新題目」表單導入 showLoadingState 攔截機制。點擊瞬間即刻鎖定觸發與返回按鈕，並替換為旋轉載入動畫，有效杜絕使用者因等待不耐煩而產生連點，確保伺服器穩定並保護 Google API 珍貴配額。

## [1.8.1] - 2026-03-13
### Added (新增)
* **混合雙引擎與多模態影音直讀**：
    * 導入 Google File API：全面升級 AI 核心架構，新增支援 .pdf 與 .mp4 / .mov 影音格式。AI 現在能具備「視覺」與「聽覺」，直接讀取 PDF 內的複雜圖表、數學公式，甚至能觀看教學影片、聽取語音並解讀簡報畫面，產出精準考題。
    * Word 純文字引擎解放：針對 .docx 檔案保留高穩定性的純文字萃取模式，並將擷取字數上限從 3 萬字大幅解放至 50 萬字，足以應付數百頁的長篇講義。

* **歷屆考題專屬模式**：
    * 刷題神器機制：於上傳教材表單新增「這是歷屆考題」核取方塊（綁定 Material 模型之 is_exam_paper 欄位）。
    * 智慧任務切換：勾選後系統將動態切換 AI Prompt，省略重點摘要生成，強制要求 AI 專注於提取文件中的考題，並針對無答案的考卷自動運算正解，大幅提升備考效率。

* **AI 錯題詳解與追蹤系統**：
    * 自動生成詳解：擴充資料庫 QuizMistake 模型新增 explanation 欄位。AI 出題時將同步產出「為何正確與為何錯誤」的深度解析。
    * 詳解視覺化介面：測驗結果報表新增專屬的「💡 AI 解析」燈泡區塊，幫助使用者真正理解錯題觀念，而不只是死背答案。
    * 原題號追蹤：錯題標題自動附加原始考卷的題號 (例如：Q3.)，方便使用者快速對照原始教材上下文。

### Changed (變更)
* **系統強健性與底層防呆機制**：
    * 中文檔名編碼防護：為徹底解決 Google GenAI SDK 底層的 ascii 中文路徑編碼報錯 Bug，實作了本機 tempfile 暫存機制。系統會在背景自動生成純英文替身檔案進行上傳，確保各類語言檔名皆能暢通無阻。
    * 雙重閱後即焚：強化隱私與空間管理，檔案交由 AI 解析完畢後，確保 Google 遠端伺服器與本機端的純英文暫存檔皆被徹底刪除，達成零殘留。
    * 攔截器白名單：修正視圖層的空檔案攔截邏輯，將影音格式加入白名單，避免無文字內容的 MP4 遭系統誤擋。

* **使用者體驗 (UI/UX) 與介面優化**：
    * 會員專區收納：重構頂部導覽列，將點數儲值、點數商城、投資儀表板與 AI 教材大腦四大功能，優雅地收納進全新的「會員專區」下拉選單中，並加入動態 Active 狀態與 fa-fw 圖示對齊，維持畫面的清爽專業。
    * 樣板渲染修復：引入 Django {% resetcycle %} 標籤，解決選擇題字母 (A, B, C, D) 跨題連續輪替導致出現 E、F 選項的顯示異常；並優化錯題卡片的 Flexbox CSS，使用 flex-grow-1 確保長文選項不溢出且左右兩欄完美等高。

## [1.8.0] - 2026-03-11
### Added (新增)
* **AI 教材大腦與無限擴充題庫**：
    * 智慧文檔解析與生成：全新建立 study_brain 模組，支援上傳 PDF 與 Word 格式教材，並自動萃取文本交由 Google Gemini API產生條理分明的 Markdown 重點摘要與結構化 JSON 選擇題。
    * 題庫無限擴充機制：實作智慧疊加演算法，對同一份教材重複點擊「訓練 AI」時，系統會自動省略摘要生成以節省 Token，並將歷史題目作為黑名單餵給 AI，強制產出全新且具深度的「情境應用題」，動態擴充題庫。

* **沉浸式學習室與追蹤系統**：
    * 互動測驗與結果報表：打造專屬學習室，實作前端選項點擊與作答邏輯，提交後自動結算成績，導向專屬「測驗結果報表」，以視覺化卡片呈現正確率、錯誤率，並提供左紅右綠的錯題比對介面。
    * 個人歷史紀錄儀表板：新增「My Quiz History」頁面，完整列表呈現使用者過去的所有測驗時間、得分與教材來源，支援一鍵點擊重溫歷史錯題，打造完整的學習閉環。

* **全站教材共享與收藏機制**：
    * 社群學習知識庫：打破單機限制，上傳的教材預設為全站公開。儀表板右側新增「我的教材庫」與「探索公開教材」雙頁籤切換功能。
